## Overview

This project refers to the Normalized Difference Vegetation Index (NDVI) calculation for my master's thesis I developed for my Master's Degree in Geographic Information Systems. The study area is Lisbon's Municipality. The NDVI was calculated using the Raster Calculator tool in ArcGIS Pro. The SENTINEL-2 MSI imagery of the study area was downloaded from Copernicus Browser (https://browser.dataspace.copernicus.eu/), 2% maximum cloud coverage, and is dated of august 18th 2018.

## Tools Used

- ArcGIS Pro
- Copernicus Browser 

## Results

The analysis resulted in a raster with the NDVI values for the Municipality of Lisbon.
